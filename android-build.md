# 📱 অ্যান্ড্রয়েড APK জেনারেশন ও কাপাসিটর বিল্ড গাইড
(Android APK Packaging & Capacitor Build Guide)

এই সার্টিফিকেট ট্র্যাকার ওয়েব অ্যাপ্লিকেশনটি **Capacitor** দ্বারা অত্যন্ত প্রফেশনাল মোবাইল-নেটিভ অ্যাপে কনভার্ট করা কাস্টমাইজড প্রোফাইল সম্বলিত। নিজের কাস্টমাইজড APK ডাউনলোড ও মোবাইলে ইনস্টল করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

---

## 🛠️ বিল্ড ও প্যাকেজিং করার শর্তাবলী (Prerequisites)
১. আপনার পিসিতে **Node.js** ইনস্টল থাকতে হবে।
২. অ্যান্ড্রয়েড প্রোজেক্ট কম্পাইল করার জন্য আপনার উইন্ডোজ/ম্যাক পিসিতে **Android Studio** এবং **Android SDK** ইনস্টল থাকতে হবে।

---

## 🚀 এপিকে (APK) তৈরি করার মেইন কমান্ডসমূহ (Step-by-Step commands)

লোকাল কোড ফোল্ডারটি ওপেন করে আপনার পিসির টার্মিনাল/পাওয়ারশেল এ পর্যায়ক্রমে নিচের ৫টি সহজ কমান্ড রান করুন:

### ১. লাইব্রেরিসমূহ ইনস্টল করুন:
```bash
npm install
```

### ২. মোবাইল ক্যাপাসিটর প্লাগইন ও লাইব্রেরিগুলো যুক্ত করুন:
```bash
npm install @capacitor/core @capacitor/cli @capacitor/android
```

### ৩. রিয়্যাক্ট-ওয়েব সাইটটিকে প্রোডাকশন অপটিমাইজড বিল্ড করুন:
```bash
npm run build
```
*(এই কমান্ডটি দেওয়ার সাথে সাথে আপনার সোর্স রুট ডিরেক্টরিতে `dist/` ফোল্ডার তৈরি হবে যা ফোনের মেইন ফাইল হিসেবে কাজ করবে)*

### ৪. ক্যাপাসিটর অ্যান্ড্রয়েড মডিউল যোগ ও সিঙ্ক করুন:
```bash
npx cap add android
npx cap sync
```

### ৫. প্রজেক্টটি অ্যান্ড্রয়েড স্টুডিওতে সরাসরি রান করুন:
```bash
npx cap open android
```

---

## 🎯 অ্যান্ড্রয়েড স্টুডিও থেকে কিভাবে APK তৈরি করবেন (.APK)
উপরের ৫ম কমান্ডটি (`npx cap open android`) দেওয়ার পর আপনার পিসিতে **Android Studio** ওপেন হবে এবং প্রজেক্টটি স্ক্রিনে লোড হবে:

১. অ্যান্ড্রয়েড মার্জিং ও গ্রেডেল সিঙ্ক (Gradle sync) সম্পূর্ণ হওয়া পর্যন্ত কয়েক সেকেন্ড অপেক্ষা করুন।
২. অ্যান্ড্রয়েড স্টুডিওর মেইন নেভিগেশন বার থেকে ক্লিক করুন: **`Build`** -> **`Build Bundle(s) / APK(s)`** -> **`Build APK(s)`**।
৩. কিছুক্ষণ বিল্ডিং প্রসেস চলার পর ডান কোণায় একটি নোটিফিকেশন আসবে- **`APK(s) generated successfully`**। সেখানে **`locate`** বাটনে ক্লিক করুন।
৪. আপনি একটি ফাইল পাবেন যার নাম **`app-debug.apk`**।
৫. এই `.apk` ফাইলটি সরাসরি আপনার ফোন মেমোরিতে নিয়ে সাধারণ মোবাইল অ্যাপের মত ইন্সটল করুন!
